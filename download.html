Install Binary Packages

Ubuntu from ubuntu.com

  1. Install in Software Center

Fedora 18 from build.opensuse.org

  1. Download this .repo file.
  2. Become root.
  3. Copy the .repo file into the /etc/yum.repos.d/ directory.
  4. Run yum install ggcov
